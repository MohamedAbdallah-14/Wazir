# New CLI flags must be registered in the option parser

When adding a new flag to a CLI command handler, always add it to the `parseCommandOptions` call (boolean or string array). If the flag is only checked inside the handler but not registered in the parser, it will never appear in the `options` object and the feature is silently unreachable.

**Recurrence:** First time, but high risk for any future CLI flag additions.

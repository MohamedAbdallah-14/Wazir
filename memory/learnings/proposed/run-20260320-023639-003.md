# The reviewer itself can trigger AP-23

Even when implementing AP-23 (enforce all workflows complete), the final review phase itself skipped the learn and prepare_next workflows. The enforcement mechanism only works at run finalization (`wazir capture summary --complete`), not during the review phase. The reviewer skill should explicitly call out all three final-review workflows (review, learn, prepare_next) as a checklist.

**Recurrence:** Happened in the very run that implemented the fix. High priority.
